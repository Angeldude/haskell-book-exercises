=== 7.5 Pattern matching

. given following declarations
+
[source]
----
k (x, y) = x
k1 = k ((4-1), 10)
k2 = k ("three", (1 + 2))
k3 = k (3, True)
----
.. type of `*k*` is `*(a, b) -> a*`
.. type of `*k2*` is `*String*` and is different from `*k1*` and `*k3*`
.. `*k3*` and `*k1*` will return `*3*`
. completed definitions
+
[source]
----
f :: (a, b, c) -> (d, e, f) -> ((a, d), (c, f))
f (a, b, c) (d, e, f) = ((a, d), (c, f))
----
